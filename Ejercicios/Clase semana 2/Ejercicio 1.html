Primera página con CSS

en esta página se inicia la base del CSS que consiste en aplicar diferentes estilos al html, aun cuando se puede utilizar el comando style, los aconcejable es crear un documento aparte con la terminación .CSS, esto permitirá que el html no tenga muchas lineas de código mejorando su forma de leer.

para enlazar el formato CSS se utilizara el comando Link, el cual debe ponerse en el head del html. esta etiqueta tendra dos carácteristicas la primera rel el cual tendra en este momento "stylesheet" y el "href" que se a usado previamente donde se indica donde estará contenida la información del CSS.

para este ejercicio se utilizo p { }, en los corchetes se pone color: orange para el color de texto y para el fondo background-color: black, se puede utilizar el nombre de colores primarios, sin embargo tambien se puede dar click al cuadro del color y configurar el color que desea aplicar.
